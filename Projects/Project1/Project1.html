Web Site Critiques

Evaluation Questions

  1. What was your first impression of this site?
  2. How does this site organize the flow of information?
  3. Does the site structure make sense or does it confuse you? Why?
  4. Does the site graphics/layout/multimedia enhance the site or detract from it? Why?
  5. How long did it take for the home page to load? Was the load time reasonable?
  6. Was the text color and size readable? Did the background enhance or detract?
  7. Did the site have a navigation bar/tool? Was it well done?
  8. Did the site have a consistent color scheme or look and feel?
  9. How does the site respond to smaller window sizes (responsive design)?
  10. What does this site offer that was unusual or very well done?

1. My first impression of this website was actually kinda impressed! I was thinking that the sites you were going to have listed were all going to be horrible sites and so when I saw the site I was suprised. It looks like a very common website format that I have seen before.

2. Regarding information flow, the site has a very long scrolling screen in each tab filled with a lot of information and different articles.

3. The site structure is well organized into categories with the tabs in the top middle, but it has so much information within each tab that isn't well structured that ends up being confusing.

4. In terms of multimedia and graphics there are many different photos all really meant to grab my attention, but it created the opposite affect for me. Instead, with there being so many pictures and clickbait things, I got a bit of Ad blindness and instead ignored all of them together.

5. The home page loaded almost instantly.

6. The readability of the text is very easy. Almost all the words are black on white background, or vice versa.

7. For navigation, the site provides a few different tabs in the top of the page to direct through categories, that is helpful, I would only wish that it would scroll so that I could refer to it easier.

8. The visual consistency and color scheme of the website was great. It is mostly black and white which is formal and seems professional.

9. When resizing the browser window to a smaller size nothing resized with it. There was no way to read or comprehend the page when it was smaller.

10. One aspect of this site that was particularly well done is how there are multiple different scrolling sections to make the page feel less like one really long screen and more like we are actually interacting with it.
